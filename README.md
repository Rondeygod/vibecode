# Discord Music Bot

Een simpele Discord muziekbot met `#` als prefix. Je kunt `#play`, `#skip`, `#pause`, etc. gebruiken. Draait op een Raspberry Pi.

## Installatie
1. Installeer Python, pip, FFmpeg
2. Installeer packages:
   pip install -r requirements.txt

3. Start de bot:
   python3 musicbot.py
